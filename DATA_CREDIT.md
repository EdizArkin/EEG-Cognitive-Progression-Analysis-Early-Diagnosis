## ⚠️ Data Set Acknowledgment and Citation ⚠️

This project utilizes Electroencephalography (EEG) and Event-Related Potential (ERP) data for the classification of Mild Cognitive Impairment (MCI) against Healthy Controls (HC), including follow-up visits.

### Source:
The primary data set is sourced from the Harvard Dataverse.

**DOI:** doi:10.7910/DVN/GDGIVG  
**Persistent URL:** https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/GDGIVG

### Required Citation:
Meghdadi, Amir, 2024, *Longitudinal EEG and ERP biosignatures of MCI*, https://doi.org/10.7910/DVN/GDGIVG, Harvard Dataverse, V1

**NOTE:** In accordance with academic best practices and the data license, this project acknowledges and cites the source of this data set in all official documents and publications.
